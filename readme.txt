Projemiz stellaris launchpad kullanarak lcd ile ekrana sayılar ve verilen isterleri yerine getirmek. 
İsterler:
1-)  Birinci butona basılması 
 Üst satırda 1’den başlayarak 9’ a kadar rakamları sırasıyla yazar. Ardından led de ışık yanar. 

 2-)  İkinci butona basılması 
Üst satırda 1’den başlayarak 9’ a kadar rakamları sırasıyla yazar. Sonrasına nokta koyar. Ardından led de ışık yanar.

 3- )Üçüncü butona basılması 
Üst satırda 1’den başlayarak 9’ a kadar rakamları sırasıyla yazar. Ardından led de ışık yanar.

4- ) Dördüncü butona basılması
Üst satırda 1’den başlayarak 9’ a kadar rakamları sırasıyla yazar. Ardından led de ışık yanar.

5- ) Beşinci butona basılması
Yazılan bütün sayıyı lcd ekranından sıfırlar. Ardından led de ışık yanar.

6- ) Altıncı butona basılması
Lcd de yazılan sayıyı paraların birimlerine bölerek alttaki kısımda kaçlık birimlere ayrıldığını yazar. Ardından led de ışık yanar.
